Add the .row class to a container with the .container class. And add any element you want with the .column class inside the container. These columns will take up the space equally. There are col-xs-*, col-sm-*, col-md-*, col-lg-* and col-xl-* available for flexible grid across mobile, tablet and desktop viewport usage.

  • col-xs-* apply to width smaller than 576px. X-Small devices (portrait phones, less than 576px)
  • col-sm-* apply to width smaller than or equal to 767px. Small devices (landscape phones, 576px up to 767px)
  • col-md-* apply to width smaller than or equal to 991px. Medium devices (tablets, 768px up to 991px)
  • col-lg-* apply to width smaller than or equal to 1199px. Large devices (desktops, 992px up to 1199px)
  • col-xl-* apply to width smaller than or equal to 1400px X-Large devices (large desktops, 1200px up to 1400).
  • col-* apply to all width, including the width wider than 1400px.

Column offset

The Flexbox grid provides margin auto utilities to set offset. There are .col-ml-auto, .col-mr-auto and .col-mx-auto to set margins between columns without using empty columns.

  1. .col-ml-auto offsets the left of the column.
  2. .col-mr-auto offsets the right of the column.
  3. .col-mx-auto offsets the left and right of the column.

Typography

Type is all set with the rems, so font-sizes and spacial relationships can be responsively sized based on a single <html> font-size property.

Fonts: The typography font base by default is Poppins served by Google. The font may be changed to Source Sans Pro by applying the class .ff-source-sans-pro to the <html> element(s). If you wish to change the default font to Source Sans Pro apply the class .ff-source-sans-pro to the <body> tag. If you would like to use both fonts interchangeably, apply the class .ff-poppins or .ff-source-sans-pro to the <html> element(s) you would like to style.

Font Size: Quickly change the font-size of text. While our heading classes (e.g., .h1â€“.h6)

Font Weights and Stylings: Change the font-weight or font-style of text with the utilities listed below. Font-style utilities are prefixed as .text-* and font-weight utilities are prefixed as .font-*.

Text-Alignment: Easily realign text to components with text alignment classes. For left, right, and center alignment, responsive classes are available that use the same viewport width breakpoints as the grid system. Text-Alignment utilities are prefixed as .text-*.

Text-Transform: Transform text in components with text capitalization classes. Text-Transform utilities are also prefixed as .text-* .

Letter Spacing: Change the letter spacing with .letter-spacing-* utilities.

Tables

Regular Tables: Properly markup the <html> with <thead> and <tbody> when building a table. Add the class .table to <table>.

Striped tables: Use the table-striped class to <table> to add zebra striped style. For hoverable table rows, you can add the .table-hover class to enable hover style.

Bordered tables: Use the .table-bordered class to <table> to add a bordered style.

Responsive Table: A responsive table will display a horizontal scroll bar if the screen is too small to display the full content. Resize the browser window to see the effect. To add this feature to your table, add a <div> wrapper to the table with the class .table-container around the table and then add the .table-responsive class to <table> element. I recommend this method for all tables to add in responsiveness!

Helper

Float: To float an element based on the current viewport size using the CSS float property. to the left apply the class .float-left and to float an element to the right apply the class .float-right.

Clear Float: To clear the float after using the above stated class. Apply the class .clearfix to the element.

Full-Width: To make element full width, apply the .full-width to the element's tag.

Margins and Paddings

For the spacing utilities, we use the same naming convention as follow:

Where property is one of:

  • m - for classes that set margin
  • p - for classes that set padding

Where sides is one of:

  • t - for classes that set margin-top or padding-top
  • b - for classes that set margin-bottom or padding-bottom .
  • l - for classes that set margin-left or padding-left, margin-right or padding-right
  • r - for classes that set margin-right or padding-right, margin-left or padding-left.
  • x - for classes that set both *-left and *-right.
  • y - for classes that set both *-top and *-bottom
  • blank - for classes that set a margin or padding on all 4 sides of the element

Where size is one of:

  • 0 - for classes that eliminate the margin or padding by setting it to 0
  • 1 - (by default) for classes that set the margin or padding to 0.5rem (8px)
  • 2 - (by default) for classes that set the margin or padding to 0.75rem (12px)
  • 3 - (by default) for classes that set the margin or padding to 1rem (16px)
  • 4 - (by default) for classes that set the margin or padding to 1.25rem (20px)
  • 5 - (by default) for classes that set the margin or padding to 1.5rem (24px)
  • auto - for classes that set the margin to auto
