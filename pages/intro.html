Start

Download

Both CSS and JS files must be downloaded and saved into respective folder before starting with For Schools.

Next steps

  1. After completing the instruction described above, create a new index.html file in the project root. 
                                    
<!doctype html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>For Schools demo</title>

        <!-- Font Awesome -->
        <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.3.0/css/all.min.css">

    </head>
    <body>
        <h1>Hello, world!</h1>
    </body>
</html>
  2. Link For Schools' CSS file through <link> tag in the <head>, and JS file through <script> tag before the closing </body>. 
                                    
<!doctype html>                                    
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>For Schools demo</title>
    <link rel="stylesheet" href="css/for-schools.css">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="scripts/for-schools.js"></script>
  </body>
</html>
  3. Hello, world! Open the page in your browser of choice to see your For Schools page.
